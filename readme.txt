== Description ==

Add-on for Event Tickets Plus 4.1+. Include ticket meta information in CSV export of event attendee list.


== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. There is no step 3!


== Frequently Asked Questions ==

= Why do I only get name and email address in my CSV export? =
Event Tickets 4.1.4 has a regression bug so that all other ticket information is left out. Use version 4.1.3 until this has been fixed.
